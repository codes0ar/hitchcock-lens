# 希区柯克变焦摄像APP — 执行计划

## 项目概述
开发一个移动端摄像APP，用户持手机从远处接近被摄人物时，通过实时人脸检测+动态变焦控制，保持人物面部在画面中大小不变，同时背景产生经典的希区柯克拉伸（Dolly Zoom / Vertigo Effect）视觉效果。

## 技术方案
- **框架**: React Native + Expo (跨平台 iOS/Android)
- **核心库**: 
  - `expo-camera` — 摄像头控制与变焦
  - `react-native-vision-camera` 或 `expo-face-detector` — 实时人脸检测
- **核心算法**: PID/比例控制，根据人脸像素尺寸偏差实时调整 zoom level

## Stage 1 — 原型验证与核心算法开发
- 加载技能: `vibecoding-general-swarm`
- 任务:
  1. 搭建React Native + Expo项目骨架
  2. 集成摄像头实时预览
  3. 集成实时人脸检测（获取人脸边界框像素尺寸）
  4. 实现核心控制算法：根据人脸大小偏差计算目标zoom值
  5. 实现zoom平滑过渡动画
  6. 添加基础录像功能
  7. 输出: 可运行的原型APP代码

## Stage 2 — 测试方案与验证标准
- 任务:
  1. 制定功能测试用例
  2. 制定性能测试标准（帧率、延迟、精度）
  3. 制定用户体验测试方法
  4. 输出: 完整测试方案文档

## Stage 3 — 优化与交付
- 任务:
  1. 根据测试反馈优化参数
  2. 完善UI界面（启动画面、录制按钮、设置等）
  3. 添加视频保存/分享功能
  4. 最终代码审查与打包
